## General info
Simple neumorphic dashboard design using react native.
	
## Technologies
Project is created with:
* react-native: 0.63.4
	
## Setup
To run this project, install it locally using npm:
```
npm install
cd ios && pod install && cd ..
react-native run-ios

```
